# Birthday Flyer Maker — Christ Apostolic Church, Hallelujah District HQ

This is your church's birthday flyer website. Follow the steps below to put it
online for free, with no coding needed.

## Deploy it online (free, ~15 minutes)

We'll use **Vercel** — a free hosting service used by millions of websites.
No credit card required.

### Step 1: Create a free Vercel account
1. Go to **vercel.com**
2. Click **Sign Up**
3. Choose **Continue with GitHub** (easiest option — you'll need a free
   GitHub account too; if you don't have one, GitHub will let you create one
   in the same flow)

### Step 2: Get this project onto GitHub
1. Go to **github.com** and log in
2. Click the **+** icon top-right → **New repository**
3. Name it something like `birthday-flyer-maker`
4. Leave it **Public** or **Private**, either works
5. Click **Create repository**
6. On the next page, click **uploading an existing file**
7. Drag in **all the files and folders** from this project (keep the folder
   structure — `src` should stay a folder, not get flattened)
8. Scroll down, click **Commit changes**

### Step 3: Deploy on Vercel
1. Back in Vercel, click **Add New** → **Project**
2. Find and select the `birthday-flyer-maker` repository you just created
3. Vercel will auto-detect it's a Vite project — leave all settings as default
4. Click **Deploy**
5. Wait about 1–2 minutes

That's it. Vercel will give you a live link like:
`https://birthday-flyer-maker-yourname.vercel.app`

Anyone with that link can open the site, build a flyer, and download it —
no login required, on phone or computer.

### Step 4 (optional): Use your own domain
If your church already owns a domain (e.g. `cachalleluyah.org`), Vercel lets
you connect it for free under **Project Settings → Domains**. You'd add
something like `flyers.cachalleluyah.org`.
